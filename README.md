Minecraft mod that changes Nether portal generation so that the return portal is the same size as the entry portal.
Primarily intented for use with Ruined Portals only / Nether Portal nerf mod. In vanilla you can always break the generated return portal and change it to your liking, not so when Ruined Portal only mod is installed!

Not compatible with mods that allow non-rectangular or flat laying portals.

Currently only Forge 1.20.1. This mod is trivial to port, give it a go!
